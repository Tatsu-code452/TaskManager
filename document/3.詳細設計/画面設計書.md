# 画面設計書（React版ワイヤーフレーム・コンポーネント設計）

## 1. ログイン画面
### ワイヤーフレーム
```
+-----------------------------+
|         ログイン            |
+-----------------------------+
| ユーザID [__________]       |
| パスワード [__________]     |
| [ ログイン ]                |
| <エラーメッセージ表示>      |
+-----------------------------+
```
### コンポーネント構成
- LoginPage
  - LoginForm
    - TextField（ユーザID）
    - TextField（パスワード）
    - Button（ログイン）
    - ErrorMessage
