# Issue（課題）API テスト仕様書（Tauri コマンド）

## 1. 対象コマンド
- get_issues
- create_issue
- update_issue
- delete_issue

# 2. テストケース

## 2.1 get_issues

| No       | 入力         | 期待結果       | チェック |
| -------- | ------------ | -------------- | -------- |
| API01-01 | project_id=1 | 課題一覧が返る | ☐        |
| API01-02 | status=Open  | Open のみ返る  | ☐        |

## 2.2 create_issue

### 正常系

| No       | 入力            | 期待結果   | JSON DB    | チェック |
| -------- | --------------- | ---------- | ---------- | -------- |
| API02-01 | title="API遅延" | 201        | 追加される | ☐        |
| API02-02 | task_id=1       | 紐付け成功 | task_id=1  | ☐        |

### 異常系

| No       | 入力         | 期待結果 | JSON DB  | チェック |
| -------- | ------------ | -------- | -------- | -------- |
| API02-03 | title=""     | 400      | 追加なし | ☐        |
| API02-04 | 不正 task_id | 400      | 追加なし | ☐        |
