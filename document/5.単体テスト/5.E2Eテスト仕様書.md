# 案件管理システム E2E テスト仕様書（Cypress）

## 1. 目的
本書は、React（フロント）＋Tauri（バックエンド）を実際に起動し、  
ユーザー視点でシステム全体が正しく動作することを確認するための  
E2E テスト仕様書である。

## 2. テスト方針
- Cypress によるブラウザ自動操作
- Tauri バックエンドも起動した状態で実施
- JSON DB をテスト用に初期化して使用
- 画面遷移・入力・保存・反映・ガント描画を確認
- 正常系・異常系を明確に分離
