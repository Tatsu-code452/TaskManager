# 案件管理システム テストデータ一覧（JSON DB）

## 1. 目的
本書は、API テスト・結合テスト・E2E テストで共通して使用する  
テストデータセットを定義する。

---

# 2. テストデータの利用方針
API テスト：テスト開始前に JSON をコピーして使用
E2E テスト：テスト開始前に JSON を初期化
結合テスト：E2E と同じデータを使用
単体テスト：必要に応じてモックデータを使用

---
