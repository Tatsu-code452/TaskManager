# プロジェクト初期化
```sh
mkdir my-app && cd my-app
npm init -y
```
# typescript導入
```sh
npm install --save-dev typescript ts-node @types/node
npx tsc --init
```
```json
{
  "compilerOptions": {
    "target": "ES2020",
    "module": "CommonJS",
    "rootDir": "src",
    "outDir": "dist",
    "strict": true,
    "esModuleInterop": true,
    "skipLibCheck": true
  }

```
# 依存ファイル、型定義導入
```sh
npm install express
npm install --save-dev @types/express

npm install pg
npm install --save-dev @types/pg
```

npm run <scripts option>
