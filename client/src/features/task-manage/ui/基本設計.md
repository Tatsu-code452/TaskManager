# 進捗管理機能

## 1. 機能概要

- タスク一覧表示
  - フェーズごとのタスクの一覧を表示する。
  - 1タスク2行とし、 1行目は計画、2行目は実績を表示する。
- 日付マトリックス表示
  - 表示範囲の期間分、日付列を表示する。
- タスク実施計画作成
  - 月日のマトリックスの計画行に、その日に作業する工数を設定する。
  - 計画は、1日単位で、マウスのD&D or 手入力にて、変更可能とする。
- タスク実施実績入力
  - 月日のマトリックスの実績行に、その日に作業した工数を設定する。
  - 1日単位で、マウスのD&D or 手入力にて、変更可能とする。
- タスク補正機能
  - 計画行の開始、終了、工数は、月日のマトリックスの入力から、自動算出する。<br>
    進捗率は、基準日までの月日のマトリックスの入力から、自動算出する。<br>
  - 実績行の開始、終了、工数は、基準日までの月日のマトリックスの入力から、自動算出する。<br>
    進捗率は、任意入力とする。<br>
    進捗率が100%でない場合は、終了は空白とする。

## 4. 画面イベント

- 初期表示
  - 表示範囲は、画面を表示した日付の、1日から末日までを表示する。
  - 基準日は、画面を表示した日付を表示する。
  - サーバのTASKテーブルから情報を取得し、タスク一覧、日付マトリックスに、表示する。
- キーボード入力
  - カーソルが当たっている項目にて、F2、または、文字入力が発生した場合、その項目を入力項目へ変更し、編集可能とする。
  - 編集完了は、EnterまたはTabを入力したときとし、入力内容にて、自動算出を行う。
  - 自動算出後、問題がなければ、サーバのTASKテーブルへの登録を行う。
