# API Demo (ログイン → CRUD の確認)

簡易デモ画面 `src/SimpleApiDemo.tsx` を使って、サーバへのログインと CRUD を試せます。

手順:

1. サーバを起動していることを確認します（例: `npm run dev` 端末でバックエンドを起動）。
2. クライアントを起動します: `npm install` → `npm run dev`（Vite）。
3. ブラウザでアプリを開き、メニューから `SimpleApiDemo` を表示するか、直接ルートに組み込んでください。
4. 「CSRFトークン取得」でトークンを取得し、「ログイン」でユーザー名/パスワードを入力してログインします。
5. 「Tasks」の「一覧取得」「作成」「削除」「更新」ボタンで CRUD を試してください。

環境変数:
- `REACT_APP_API_BASE` を設定すると API のベース URL を変更できます（デフォルト: `http://localhost:3000/api`）。

注意:
- サーバの API 実装により、作成/更新に必要なフィールドや検証が異なります。エラーはポップアップで通知されます。
